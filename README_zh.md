[简体中文](README_zh.md) | [ENGLISH](README.md)

这是一款深度绑定[NeHex](https://github.com/nehex)个人博客引擎的个人小页.  

如果想要使用，请先安装使用[NeHex](https://github.com/nehex)

# 使用
clone 到本地之后，复制`.env.example` 为 `.env` 并编辑其中内容

使用`npm run dev` 来开发
